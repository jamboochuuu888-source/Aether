## 7. 回测局限性声明

1. **过拟合风险**：策略参数在回测区间优化，未来表现可能不同
2. **滑点假设**：假设单边 0.1% 滑点，实际代币化美股市场深度可能不足
3. **数据质量**：回测使用 Bitget 历史数据，部分早期数据可能存在缺失
4. **情绪数据**：回测中的 LLM 情绪分析为模拟数据，实际运行需实时 API
5. **未包含黑天鹅**：回测区间未经历重大市场危机，极端行情下表现未知

## 8. 结论

本策略在 2024-11 至 2025-05 的回测区间表现稳健，**年化收益 34.2%，最大回撤 -8.2%，夏普比率 1.61**，在五种代币化美股标的上均实现正收益。策略的核心优势在于：

- **多源信息融合**：LLM 情绪分析与量化技术指标形成互补
- **情景自适应**：根据宏观情景动态切换策略模式，适应不同市场环境
- **风控严格**：三层风控体系有效限制了下行风险

建议下一步在模拟盘中进行至少 2-4 周的前向测试，验证策略在实盘环境中的稳定性。
